# FairSplit AI User Manual

## Release Status

`V1.0` initial version

FairSplit AI `V1.0` is the initial public version of the product. This release is focused on split-first budgeting, shared-expense clarity, and optional trip-draft planning. Forgot-password support is planned for a later update and is not included in the `V1.0` user flow.

## What FairSplit AI Is For

FairSplit AI helps groups quickly understand:

- who paid
- who participated
- how much each person owes
- what the cleanest settlement plan looks like

Trip planning is included as a supporting feature, but expense splitting remains the primary workflow.

## Main Features In V1.0

- Account signup and login
- Natural-language expense splitting
- Saved split history
- Trip draft generation and storage
- Reviews and ratings
- Live activity updates in supported views
- Admin dashboard access for seeded admin accounts

## Features Planned For Later Updates

- Forgot-password recovery flow

## Login And Access

Users can access FairSplit AI with:

- email and password
- supported social login providers when configured

For seeded local or managed environments, the app can also include:

- admin login
- demo student login

## Typical User Flow

1. Sign up or log in.
2. Enter an expense scenario in plain language.
3. Review the calculated shares and reimbursements.
4. Save and revisit split history when needed.
5. Optionally generate a trip draft for budget planning.

## Trip Planning Position In The Product

Trip planning is intentionally secondary in `V1.0`. It is available to help users prepare travel budgets and save plan drafts, but the main product objective remains fair and simple expense splitting.

## Notes For V1.0

- This release is centered on the core split experience.
- Some extended account-management features are reserved for future updates.
- Forgot-password support will be added in a later version.
